Privacy Policy

Last updated: 21 March 2026

What this application does

Channel Analyser is a personal-use tool that connects to the eBay API to analyse listing performance. It is not a public service and does not collect data from third parties.

Data collected

This application accesses only the eBay account data of the authenticated user (listing titles, views, impressions). No personal data is collected, stored, or shared with any third party.

Data storage

All data is processed locally on the user's own machine. No data is transmitted to external servers beyond the eBay and Anthropic APIs required for the tool to function.

Cookies

This application does not use cookies.

Contact

For questions about this policy, contact the repository owner via the project's issue tracker.
